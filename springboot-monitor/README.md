springboot-monitor
=================

1. 基于`springboot 1.5.x`，`springboot-admin 1.5.x` 提供监控服务端
2. 由于springboot2.x 需要采用对应的springboot-admin 2.x 

客户端示例：[springboot-docker](https://github.com/cjp1016/springcloud-examples/tree/master/springboot-docker)
